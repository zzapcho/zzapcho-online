# 제품 개요

## 프로젝트 목표

`zzapcho-online`은 잡초 약탈서버 전용 Minecraft 런처입니다. 기존 `zzapcho-launcher`를 기반으로 하되, 서버 선택형 런처가 아니라 `online.zzapcho.kr` 단일 서버 입장에 맞춘 제품으로 전환합니다.

## 대상 사용자

- 잡초 약탈서버에 접속하는 일반 플레이어
- 모드/리소스팩 구성을 GitHub에서 관리하는 운영자

## 핵심 기능

- Microsoft 계정 로그인
- 공식 manifest 기반 클라이언트 파일 동기화
- SHA-256 검증
- 비공식 파일 quarantine
- 사용자 셰이더 추가/삭제/Modrinth 설치
- 서버 상태와 접속 인원 표시
- GitHub Releases 자동 업데이트

## 제거한 기능

- 프리셋 선택
- 임의 서버 입력
- 사용자 모드/리소스팩 추가
- Modrinth 검색/다운로드
- 사용자 파일 stash/restore

## 디자인 방향

차콜 배경, 아이보리 텍스트, 샴페인 골드 포인트를 사용합니다. 약탈서버 전용 런처답게 과장된 네온/RGB나 저가형 게임런처 느낌을 피하고, 단순하고 신뢰감 있는 도구형 UI를 유지합니다.
